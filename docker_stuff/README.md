# RPCOS4Ph2 - SWATCH Based

## Dev env

### Login

`docker login gitlab-registry.cern.ch`


### Build

`docker build --tag ftorresd/swatch_build_env .`

### Run

`docker run -it ftorresd/swatch_build_env:latest`
